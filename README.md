# The Lemniscate Principle

**Saddle-Point Dynamics and the Universal Topology of Adaptive Transformation**

Interactive computational model and academic paper by **Jennifer Leigh West** (2026).

## Contents

- **lemniscate_v3.html** — Interactive web version with Lyapunov stability theory applied to bistable adaptive systems
- **West_Lemniscate_Principle_v3.pdf** — PDF of the academic paper

## Abstract

The Lemniscate Principle: the moment of greatest consequence is the moment of least visibility. This repository hosts the interactive proof and paper materials.

## View online

After deployment, the HTML paper can be viewed via GitHub Pages or by opening `lemniscate_v3.html` locally.
